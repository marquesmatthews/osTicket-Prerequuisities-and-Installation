# 🛠️ osTicket: Post-Installation Configuration

## 📌 Project Overview

In this project, I configured the osTicket Help Desk system after installation. This included creating roles, departments, teams, agents, users, SLA plans, and Help Topics to simulate a real-world IT Help Desk environment.

---

## 🎯 Objectives

- Configure Roles
- Configure Departments
- Configure Teams
- Create Help Desk Agents
- Create Users
- Configure SLA Plans
- Configure Help Topics
- Prepare the Help Desk environment for ticket management

---

## 🖥️ Technologies Used

- Microsoft Azure
- Windows 10 Virtual Machine
- Remote Desktop (RDP)
- osTicket
- IIS
- MySQL
